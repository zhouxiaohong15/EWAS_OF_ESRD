Genome-wide DNA methylation analysis revealed epigenetic mechanism underlying end stage renal disease


This page contains the main code and example dataset for the EWAS and meta-EWAS analysis in the manuscript:


Requirement
R (version 3.50.3)



example_data: This folder contains the example methylation data.
Each.cohort.EWAS:This folder contains the code  for running the EWAS analysis in each cohort.
Meta.EWAS: This folder contains the code  for running the meta EWAS analysis.
